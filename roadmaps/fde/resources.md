# FDE Free Resources

Everything the [roadmap](roadmap.md) links to, plus alternatives. All free (free tier at most). One primary per topic — don't collect courses, finish them.

## Python
- **[CS50P — Harvard's Intro to Programming with Python](https://cs50.harvard.edu/python/)** — the best free structured Python course; problem sets force real practice. *(~30 hrs used selectively; skip lectures you know)*
- Alt: [Python official tutorial](https://docs.python.org/3/tutorial/) — terse, good if you already program.

## Git
- **[Learn Git Branching](https://learngitbranching.js.org/)** — interactive visual Git; main + remote tracks are enough. *(~3 hrs)*

## SQL
- **[SQLBolt](https://sqlbolt.com/)** — fastest zero-to-JOINs interactive tutorial. *(~4 hrs)*
- **[SQL Practice](https://www.sql-practice.com/)** — free query drills against realistic schemas. *(~4 hrs)*
- Alt: [SQLZoo](https://sqlzoo.net/wiki/SQL_Tutorial).

## DSA
- **[NeetCode Roadmap](https://neetcode.io/roadmap)** — pattern-ordered problems with the best free video explanations; free tier is plenty. Do ~25–40 problems across the first 6 patterns; FDE coding rounds are practical, so don't grind past that. *(~20 hrs)*
- Alt: [roadmap.sh DSA roadmap](https://roadmap.sh/datastructures-and-algorithms) for a topic map.

## APIs & backend
- **[FastAPI tutorial](https://fastapi.tiangolo.com/tutorial/)** — the framework AI teams actually use; docs double as a course. *(~6 hrs)*
- **[Real Python — API Integration](https://realpython.com/api-integration-in-python/)** — consuming APIs properly: auth, pagination, retries. *(~2 hrs)*

## LLM fundamentals & prompting
- **[Anthropic documentation](https://docs.anthropic.com/)** — get-started, prompt engineering, tool use, evals; free API tier for exercises. *(~6 hrs reading + practice)*
- **[Anthropic courses (GitHub)](https://github.com/anthropics/courses)** — free interactive notebooks: API fundamentals, prompt engineering, tool use, evals. *(~8 hrs)*
- **[OpenAI Cookbook](https://cookbook.openai.com/)** — worked examples incl. function calling; know both providers. *(~3 hrs selectively)*
- **[Hugging Face LLM Course](https://huggingface.co/learn/llm-course)** — chapters 1–2 for how transformers/tokenizers work. *(~5 hrs)*
- Alt: [Prompting Guide](https://www.promptingguide.ai/) — provider-neutral prompting reference.

## RAG, agents, MCP
- **[DeepLearning.AI short courses](https://www.deeplearning.ai/short-courses/)** — free ~1.5 hr courses; pick one RAG and one agents course. *(~3 hrs)*
- **[Hugging Face Agents Course](https://huggingface.co/learn/agents-course)** — agent fundamentals + frameworks + agentic RAG. *(~6 hrs for units 1–2)*
- **[Model Context Protocol](https://modelcontextprotocol.io/)** — intro + quickstart server; named in Anthropic FDE postings. *(~3 hrs)*
- Alt: [LangChain Academy](https://academy.langchain.com/) — free LangGraph course if you want a framework deep-dive.

## Evals (highest-leverage topic per hour)
- **[Hamel Husain — Your AI Product Needs Evals](https://hamel.dev/blog/posts/evals/)** — the canonical practitioner essay. *(~1 hr)*
- Anthropic docs evals section (linked above) — success criteria and eval-set design.

## System design
- **[System Design Primer (GitHub)](https://github.com/donnemartin/system-design-primer)** — core concepts section only: caching, queues, load balancing, DB scaling. *(~6 hrs)*
- **[Excalidraw](https://excalidraw.com/)** — free whiteboard; practice every design here since interviews use one.
- [Exponent blog](https://www.tryexponent.com/blog) — free articles incl. LLM system design and the FDE guide.

## Interview intel & practice
- **[Exponent — FDE Interview 2026 Guide](https://www.tryexponent.com/blog/forward-deployed-engineer-interview-the-definitive-2026-guide-fde)** — the best single free FDE interview writeup. *(~1 hr)*
- **[Pramp](https://www.pramp.com/)** — free peer-to-peer mock interviews. *(2 × 1 hr)*
- **[Palantir blog](https://blog.palantir.com/)** — FDE model and open-ended interview guidance from the source.
- **[Glassdoor Palantir FDSE interview reports](https://www.glassdoor.com/Interview/Palantir-Technologies-Forward-Deployed-Software-Engineer-Interview-Questions-EI_IE236375.0,21_KO22,56.htm)** — real recent questions (free account required).
- **[The New Stack — Forward Deployed Engineers](https://thenewstack.io/forward-deployed-engineers-ai/)** — why labs are hiring FDEs; good narrative for "why this role" answers.

## Free certifications (résumé line items)

Certificates don't replace projects, but they get past résumé filters and show initiative. All genuinely free:

- **[Hugging Face Agents Course certificate](https://huggingface.co/learn/agents-course)** — pass the final GAIA-based challenge; the most credible free GenAI cert right now for FDE-type roles.
- **[Databricks Academy — free self-paced trainings](https://www.databricks.com/learn/training/home)** — the free Generative AI / Lakehouse Fundamentals accreditations; directly relevant if you target Databricks FDE.
- **[Kaggle Learn certificates](https://www.kaggle.com/learn)** — Python, Pandas, Intro ML micro-course certificates; quick wins in week 1–2.
- **[freeCodeCamp certifications](https://www.freecodecamp.org/learn)** — Data Analysis with Python / Machine Learning with Python; substantial (each ~300 hrs claimed, faster in practice) and fully free.
- **[Elements of AI](https://www.elementsofai.com/)** — University of Helsinki; free certificate; light technically but a fine first line item.
- **[Google Cloud Skills Boost — intro GenAI courses](https://www.cloudskillsboost.google/)** — several introductory generative-AI badges are free; useful if a target company is GCP-based.
- **[Microsoft Learn — AI training paths](https://learn.microsoft.com/en-us/training/)** — free modules and badges (Azure AI fundamentals learning path; the exam itself is paid, the training isn't).

Priority for FDE: HF Agents cert → Databricks GenAI Fundamentals → Kaggle certs. Skip cert-collecting beyond that — a deployed project with an eval table beats five more badges.
